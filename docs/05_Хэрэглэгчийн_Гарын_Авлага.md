# Хэрэглэгчийн гарын авлага

## 1. Төслийг ажиллуулах

1. Virtual Environment идэвхжүүлэх
2. Вебкам илрүүлэлт ажиллуулах
   python src/detect_realtime.py
3. q дарж гарна.

## 2. Өөрийн загвар сургах

1. Зургаа data/images/train болон data/images/val фолдерт хуулах
2. LabelImg ашиглан тэмдэглэгээ хийх
3. data/custom.yaml тохиргоог засах
4. Дараах командыг ажиллуулах:
   python src/train_model.py

## 3. Үр дүн

Сургалтын дараа runs/detect/train/weights/best.pt файл үүснэ.
