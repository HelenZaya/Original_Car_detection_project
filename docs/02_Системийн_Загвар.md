# Системийн Загвар

## Архитектур

1. Вебкамераас дүрс авах (Input)
2. Frame боловсруулах (OpenCV)
3. YOLOv8 загвараар танилт хийх (Model)
4. Үр дүнг дэлгэцэнд харуулах (Output)

## Гол бүрэлдэхүүнүүд

- Input Module
- Model Module
- Detection Module
- Output Module

## Өгөгдлийн урсгал

Camera → OpenCV → YOLOv8 Model → Annotated Output → Display
