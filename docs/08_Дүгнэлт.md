# Дүгнэлт

Энэхүү төслөөр Python болон YOLOv8 ашиглан бодит цагийн объект таних системийг амжилттай хэрэгжүүллээ.  
Систем нь урьдчилан сургагдсан болон өөрийн dataset ашиглах чадвартай, бодит орчинд вебкам ашиглан ажилладаг.

## Цаашдын хөгжүүлэлт

- Нүүр таних модуль нэмэх
- Объект хянах систем (Tracking, DeepSORT) нэвтрүүлэх
- Streamlit/Flask ашиглан вэб интерфэйс хийх
