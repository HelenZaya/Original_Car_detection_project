# Төслийн Тойм

## Төслийн нэр

YOLOv8 болон OpenCV ашиглан реал-тайм объект таних систем

## Зорилго

Вебкамераас ирж буй дүрс дээрх объектуудыг автоматаар таних, байршлыг илрүүлэх, ангилах систем хөгжүүлэх.

## Хүрээ

- Хүн, машин зэрэг нийтлэг объект таних
- Өөрийн dataset ашиглан шинэ объект сургалт хийх
- Реал-тайм танилтын орчин үүсгэх

## Ашигласан технологиуд

| Ангилал   | Технологи                  |
| --------- | -------------------------- |
| Хэл       | Python                     |
| Framework | PyTorch                    |
| Номын сан | OpenCV, Ultralytics        |
| Орчин     | Virtual Environment (venv) |

## Хүлээгдэж буй үр дүн

Бодит цагийн орчинд ажиллах, вебкамераас орж ирж буй дүрсэн дээр объект таних чадвартай Python систем.
